🚀 Instalación Local

1. Extraer archivos

tar -xzf salsamixlive-code.tar.gz cd salsamixlive-backup

2. Backend (Python)

  1. Instalar dependencias:
    cd backend && pip install -r requirements.txt
  2. Configurar MongoDB local o usar el actual
  3. Editar backend/.env con tus configuraciones
  4. Iniciar servidor:
    uvicorn server:app --host 0.0.0.0 --port 8001

3. Frontend (React)

  1. Instalar dependencias:
    cd frontend && yarn install
  2. Editar frontend/.env con la URL de tu backend
  3. Iniciar desarrollo:
    yarn start
  4. O crear build de producción:
    yarn build

4. Variables de Entorno Importantes

Backend (.env):

  • MONGO_URL - URL de MongoDB
  • DB_NAME - Nombre de la base de datos
  • JWT_SECRET_KEY - Clave secreta para JWT
  • STREAM_URL - URL del stream de Asura Hosting

Frontend (.env):

  • REACT_APP_BACKEND_URL - URL del backend
  • REACT_APP_STREAM_URL - URL del stream
